# Clinician Notes Folder

This folder contains transcriptions from the clinician notes transcription feature.

These transcriptions are created when users use the `/transcription` page to convert speech to text for clinical documentation.

## JSON Format

Each transcription file follows this format:

```json
{
    "ClientId": "10003",
    "ConversationId": "0a7b112e-3e59-4132-82a9-b399f782e859",
    "StartTime": "2024-11-22 03:00:00",
    "EndTime": "2024-11-22 03:14:00",
    "Content": "Customer: Hello, I need help with..."
}
```
