# Cybersecurity Support Agent Transcriptions

This folder contains transcriptions from cybersecurity support agent conversations.

These transcriptions are created when callers interact with the AI-powered voice agent
that provides cybersecurity incident support (via phone calls or web interface).

## JSON Format

Each transcription file follows this format:

```json
{
    "ClientId": "10003",
    "ConversationId": "0a7b112e-3e59-4132-82a9-b399f782e859",
    "StartTime": "2024-11-22 03:00:00",
    "EndTime": "2024-11-22 03:14:00",
    "Content": "Agent: Good day, thank you for calling Contoso Inc...\n\nCustomer: Hi, I've been having some network issues..."
}
```

## Agent Personas

The cybersecurity support agent uses different personas (Sarah, Mike, Aroha, Tane, Emma)
which will be reflected in the transcription content.
